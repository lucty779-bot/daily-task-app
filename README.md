# 每日任务 - Web 版 Demo 📱

极简高效的每日任务管理应用 - Web/PWA 版本

## 🚀 快速开始

### 方式 1：直接打开
直接用浏览器打开 `index.html` 即可使用！

### 方式 2：本地服务器（推荐）
```bash
cd /home/admin/.openclaw/workspace/daily_task_web
python3 -m http.server 8080
```
然后访问 http://localhost:8080

### 方式 3：使用 npx
```bash
npx serve .
```

## 📋 功能特性

### ✅ 已完成
- [x] **今日任务清单** - 按早/中/晚/睡前分段显示
- [x] **四象限优先级** - 紧急/重要/一般/日常
- [x] **番茄钟专注** - 25 分钟工作 + 5 分钟休息
- [x] **完成率统计** - 今日/本周/分类统计
- [x] **任务管理** - 添加/完成/删除/筛选
- [x] **深色模式** - 支持明暗主题切换
- [x] **数据持久化** - localStorage 本地存储
- [x] **数据导出/导入** - JSON 格式备份
- [x] **PWA 支持** - 可安装到手机桌面

### 🎯 核心功能

1. **今日任务** - 主页显示当天任务，按时间段分组
2. **任务清单** - 所有任务管理，支持筛选（全部/未完成/已完成）
3. **统计** - 完成率环形图、总体统计、分类统计
4. **我的** - 设置（通知、深色模式、早 8 点提醒）、数据管理

## 📁 文件结构

```
daily_task_web/
├── index.html        # 主页面（13KB）
├── styles.css        # 样式表（12KB）
├── app.js            # 应用逻辑（15KB）
├── manifest.json     # PWA 配置
└── README.md         # 说明文档
```

**总代码量**: 约 40KB，单文件即可运行！

## 🎨 设计特点

- **极简界面** - 底部 4 Tab 导航（今日/任务/统计/我的）
- **流畅动画** - 任务完成、进度环、番茄钟计时
- **响应式** - 适配手机/平板/桌面
- **深色模式** - 保护视力，夜间友好

## 💾 数据存储

所有数据存储在浏览器 localStorage：
- 任务数据
- 用户设置
- 番茄钟计数

**数据导出**: 设置 → 导出数据 → 下载 JSON 备份
**数据导入**: 设置 → 导入数据 → 选择备份文件

## 📱 PWA 安装

### Chrome/Edge
1. 访问应用
2. 地址栏出现安装图标
3. 点击安装

### Safari (iOS)
1. 访问应用
2. 点击分享按钮
3. 选择"添加到主屏幕"

### Android
1. 访问应用
2. 菜单 → 添加到主屏幕

## 📝 使用技巧

1. **快速添加任务**: 点击右下角 + 按钮
2. **完成任务**: 点击任务左侧圆圈
3. **删除任务**: 在任务清单页面点击 🗑️
4. **筛选任务**: 任务页面右上角选择筛选条件
5. **开始番茄钟**: 今日页面点击番茄钟卡片
6. **切换主题**: 我的 → 深色模式
